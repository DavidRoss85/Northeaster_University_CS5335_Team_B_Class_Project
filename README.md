# Northeaster University CS5335 Team B Class Project

## Project Backup
This repository contains the full ROS 2-based semantic mapping project developed by **Team B** for the CS 5335 Final Project. It includes all modules for perception, mapping, object localization, and visualization, along with the code used to run the TurtleBot 4 demo.  
This backup is shared for team collaboration and version control across development machines.
